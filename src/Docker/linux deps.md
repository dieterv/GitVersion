# Linux dependencies

**Alpine**
`apk add libgcc libstdc++ krb5-libs icu-libs`

**Centos 7 & 8**
`yum install -y icu`

**fedora 33**
`dnf install -y icu`

**debian 9 && 10**
`sudo apt-get install -y libgssapi-krb5-2 libicu-dev`

**opensuse 15.2**
`zypper install -y libicu-devel`

**ubuntu 16.04, 18.04, 20.04**
`none`
